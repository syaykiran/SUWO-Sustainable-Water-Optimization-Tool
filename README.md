#   SUWO - Sustainable Water Optimization Tool

##   Description

**SUWO (Sustainable Water Optimisation Tool)** is an open-source, Python-based simulation-optimization framework designed for basin-scale integrated water resources management (IWRM). SUWO simulates and optimizes water allocation within the water-energy-food-ecosystem (WEF-E) nexus. It considers various purposes under different hydrological conditions, such as irrigation, hydropower generation, and ecological flow requirements. By utilising a multi-objective genetic algorithm (MOGA), SUWO generates Pareto-optimal solutions, enabling comprehensive trade-off analyses among competing water demands and objectives. The model's Excel-based interface facilitates user-friendly data input and supports scenario-based planning. SUWO empowers decision-makers to explore and evaluate optimal water allocation strategies, promoting sustainable water resource management in complex river basins facing diverse and competing demands.

##   Features

* Integrated simulation-optimization model for basin-scale water resource management.
* Multi-objective genetic algorithm (MOGA) for generating Pareto-optimal solutions and trade-off analysis.
* Excel-based interface for user-friendly data input.
* Simulation of reservoir operations, irrigation management, hydropower generation, and ecological flow constraints.
* Evaluation of management, climate, and environmental scenarios.
* Open-source code in Python for transparency and collaboration.
